Adicionar estilo mais fácilmente

adicionar estilo da forma acima (um por um) é muito trabalhoso, por isso podemos usar o arquivo .css

no arquivo style.css, adicionamos uma tag (por exemplo p de paragráfo), e adicionamos chaves({}) qualquer declaração entre as chaves, será aplicada para todos os elementos p (que foi o que usamos no exemplo)

p{
declaração aqui (ex: color: blue;)
}

ah, e podemos usar quantas declarações quisermos, ex:

p{
color: blue;
background-color: black;
font-family: helvetica;
font-weight: bold;
}

se quisermos personalizar a página toda, podemos usar o seletor body{}

vinculando a folha de estilo (style.css)

para vincularmos a folha de estilo usamos a tag link:
《link rel="stylesheet" href="style.css"》
